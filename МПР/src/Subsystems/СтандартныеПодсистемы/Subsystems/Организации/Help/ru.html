Организации

Торговое предприятие может состоять из нескольких организаций, от имени которых будут оформляться документы. В программе ведется учет товародвижения в разрезе организаций торгового предприятия. Для каждой организации определяется список касс каждого магазина (операционных и ККМ), банковских счетов, система налогообложения:общая или упрощенная.

Список организаций открывается по ссылке НСИ – Предприятие – Организации. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные организации, задавая поиск по значениям различных реквизитов. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные элементы справочника, задавая поиск по значениям различных реквизитов. Любой адрес, введенный в программе, можно посмотреть на картах Яндекс.Карты или Google Maps. Для этого в контекстном меню поля ввода адреса (или гиперссылки с адресом) нужно выбрать пункт Адрес на Яндекс.Карты или Адрес на Google Maps. Подробнее смотри – Организации.
