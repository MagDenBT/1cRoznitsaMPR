Возвраты товаров поставщикам

Документ Возврат товаров поставщику применяется для оформления возвратов внешней поставки.

В зависимости от настроек магазина могут применяться две схемы оформления возврата:

· Однофазная – менеджер оформляет документ Возврат товаров поставщику, согласно которому уменьшаются остатки товара на складе магазина;

· Двухфазная (ордерная) – менеджер вводит в систему документ Возврат товаров поставщику, в котором регистрирует информацию по ценам и количеству возвращаемого товара, но фактически отгруженное со склада количество товара регистрирует кладовщик с помощью документа Расходный ордер.

Список документов открывается по ссылке Закупки - Закупки - Возвраты товаров поставщикам. Для удобства работы Пользователь может отфильтровать список документов по поставщику, магазину, складу выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов. Дополнительные команды доступны через кнопку Еще. Для документа предусмотрены следующие печатные формы:

· Накладная - свободная форма,

· Накладная (с розничными ценами) - свободная форма,

· Счет фактура.

· ТОРГ-12 (Товарная накладная на возврат) - форма регламентного учета,

· УПД (Универсальный передаточный документ).

· 1-T – Товарно - транспортная накладная.

Новый документ можно добавить с помощью кнопки Создать в командной панели списка. Возврат товаров поставщику также можно оформить на основании определенного документа Поступление товаров.

Заполнение шапки документа

В шапке нового документа нужно заполнить следующие поля:

· Поставщик, которому возвращается товар, выбирается из списка контрагентов;

· Магазин, из который возвращается товар;

· Склад, с которого будет отгружаться товар, по умолчанию - склад поступления, указанный для данного магазина;

· Организация, по умолчанию подставляется организация склада.

Некоторые из этих полей заполняются автоматически в следующих случаях:

для пользователя, создающего документ, соответствующие значения настроены по умолчанию;

перед созданием нового документа в списке документов были установлены значения отбора;

поступление формируется на основании документа Поступление товара.

Комиссионная торговля

В случае возврата комиссионного товара заполняется реквизит "Договор" и по регистру накопления "Товары организация" производится расход товара в разрезе измерений "Поставщик" и "Договор"

ЭДО

В подменю «ЭДО» есть команды создания и отправки электронного документа.

Созданный на основании возврата счет-фактура отправляется отдельно из формы документа.

Заполнение табличной части

Если документ Возврат товаров поставщику формируется на основании документа Поступление товаров, то табличная часть, в том числе и цены, заполняется из документа-основания.

Добавить товары в табличную часть можно следующими способами:

· с помощью кнопки Добавить командной панели табличной части документа;

· с помощью сервиса Подобрать товары, который запускается из командной панели табличной части;

· по штрихкоду. Чтобы добавить товар по штриховому коду, нужно перейти в табличную часть и считать штриховой код сканером. Кроме того, можно нажать кнопку Поиск по штрихкоду в командной панели или горячую клавишу F7 и ввести символы штрихкода вручную или с помощью сканера в открывшемся окне. Если данный штрихкод зарегистрирован в системе, то соответствующий ему товар будет добавлен в табличную часть. В противном случае в правой части формы появится сообщение об ошибке. Если товар с таким штрихкодом уже введен в табличную часть, то строка с данным товаром будет выделена, а количество увеличится на 1.

Для каждой товарной строки указывается:

· Характеристика номенклатуры – можно выбрать из списка характеристик данной номенклатуры;

· Упаковка – по умолчанию указывается основная единица измерения, но может быть выбрана любая упаковка данной номенклатуры;

· Количество – количество возвращаемых единиц или упаковок;

· Цена – цена возврата;

· Ставка НДС – заполняется по указанной для данной номенклатуры ставке НДС.

Ввод серии товара

Серию товара необходимо вводить в строке табличной части документа Возврат товаров поставщику при выполнении нескольких условий: на предприятии ведется учет товара по сериям, для данного вида номенклатуры настроена политика учета серий, согласно этой политике нужно учитывать серии при отгрузке товаров по возврату поставщику; схема возврата товаров для данного магазина однофазная. В этом случае в строке товара в поле Количество будет отображен значок Статусы серии. Значок красного цвета означает, что серии еще не введены. Для ввода серии нужно из поля Количество открыть форму Ввод количества и серий товаров и заполнить её. После корректного ввода серии и количества, значок серии меняет свой цвет на зеленый.

Дополнительные реквизиты документа

На странице Дополнительно устанавливаются флаги в разделе Налогообложение.
Включенный по умолчанию флаг Учитывать НДС может быть снят вручную, при этом сумма документа рассчитается без учета НДС. Признак Цена включает НДС влияет на значение цены, отображаемой в табличной части документа, и на алгоритм расчета суммы документа.

Оформление поступления денежных средств от поставщика при возврате товаров

Поставщик может вернуть деньги за возвращенные ему товары непосредственно в операционную кассу магазина или безналичным платежом.
Возврат денег оформляется, соответственно, документами Приходный кассовый ордер или Регистрация безналичной оплаты с установленным видом операции Возврат денежных средств от поставщика.

Внесение денег от поставщика можно оформить на основании документа Возврат товаров поставщику. В этом случае сумма создаваемого документа рассчитывается автоматически как сумма документа-основания за вычетом уже проведенных платежей.

Причины возвратов поставщикам

Анализ причин возвратов товаров поставщикам можно провести с помощью отчета Анализ причины возвратов поставщикам.

Возврат алкогольной продукции

Возврат алкогольной продукции поставщику на основании поступления формируется точно также, как и для любых других товаров. В меню Создать на основании нужного документа Поступление товаров необходимо выбрать пункт Возврат поставщику и в открывшейся форме откорректировать табличную часть: изменить количество или удалить строки. После проведения документа станет доступна команда Передать в ЕГАИС.

После выполнения команды статус документа изменится на Передан в ЕГАИС.

Возврат табачной продукции поставщику

Для оформления возврата табачной продукции используется документ Возврат товаров поставщику. В интерфейсе пользователя документы данного вида расположены в разделе Закупки.

В документе данного вида можно осуществлять сканирование, как в ТЧ документа, так и в форме Подбор и проверка табачной продукции. При работе в форме Подбор и проверка табачной продукции для отсканированной позиции сразу будет установлен Статус проверки, который при необходимости можно изменить вручную.

При включенной опции Контролировать статусы кодов маркировки в момент сканирования выполняется онлайн проверка кодов в ИС МОТП. Если данная проверка не будет пройдена, то товар будет запрещен к возврату.

После сканирования позиций необходимо нажать кнопку Завершить подбор, после чего будет осуществлен контроль количества проверенных элементов. Если товар имел статус Отложена или Не проверялась, то программа задаст уточняющий вопрос об установке статуса товара. Если выбрать вариант В наличии, то товар будет отражен в ТЧ документа. Если же выбрать вариант Отсутствующие, то товар в ТЧ перенесен не будет.

Примечание: Возвращать блоки, принятые созданным вручную документом Поступление товаров возможно только с включенной опцией Контролировать статусы кодов маркировки.

Чтобы отправить электронный УПД с кодами маркировки контрагенту необходимо в открытой форме проведенного документа Возврат товаров поставщику нажать кнопку ЭДО и выбрать команду Создать электронный документ. В открывшемся электронном документе необходимо нажать кнопку Подписать и отправить. Возможно потребуется ввести пароль закрытого ключа сертификата и нажать кнопку Подписать.

После электронного подтверждения получения товаров контрагентом в ИС МОТП для каждой марки будет зарегистрирована информация о смене владельца и статуса на значение Введен в оборот, возвращен.
